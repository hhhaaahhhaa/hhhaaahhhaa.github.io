Eating Bot

    Eating Bot is a program about deciding dinner. It's a hybrid recommendation system from machine learning methods and rule-based methods such as Markov process and distribution optimizing algorithms. Users are required to register their data before using it.

Eating Bot's CLI (Command-Line Interface) supports several commands and two modes: single mode/ group mode.

Language: Python (Pytorch)
